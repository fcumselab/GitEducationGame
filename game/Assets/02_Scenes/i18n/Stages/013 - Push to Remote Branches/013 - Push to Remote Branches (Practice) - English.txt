StageTitle,"Creating and Pushing Remote Branches"
Story Background/Content/Story,"You are a student in the latter half of the 'Project Management' course. 
The teacher has asked students to form groups and develop an application on a Git service platform.

You and your team members decided to develop a 'Snake Game' project. 
The team will apply version control practices to boost development efficiency.

Your responsible task is to write the 'README' file. 
A short while ago, you tried writing some test text on the created remote branch.

Now, please try creating a new branch to complete the introduction of the Snake Game, 
and push it to the remote repository. 
Additionally, please delete the remote branch used for testing to keep the remote repository clean."
Stage Quest/1,"Open the 'Commit History' window"
Stage Quest/2,"Create and switch to a new branch locally"
Stage Quest/3,"Add the Snake Game introduction on the 'update-readme' branch"
Stage Quest/4,"Continue completing the Snake Game introduction
on the 'update-readme' branch"
Stage Quest/5,"Create a new commit 
and update the 'Commit History' window"
Stage Quest/6,"Push the 'update-readme' branch to the remote repository
and update the 'Commit History' window"
Stage Quest/7,"Delete the 'test-readme' branch from both the local and remote repositories
and update the 'Commit History' window"

Q2（提示）：
If you are unsure how to verify whether the Git system is installed on your computer,
here are some hints to help you reach your objective:

1. Please make sure that you have already opened the 'Command Line' window.

2. Locate the 'input command field';
   this is where you can type and execute your commands.

3. Among Git commands,
   there is a specific command to check if version control is installed.
   Please look it up in the Game Manual for details.

4. When typing the command,
   make sure the letter case is correct, and separate them with spaces.
   Click the arrow icon on the right or press the Enter key on your keyboard to execute it.

5. You can check the Git Commands topic in the Game Manual,
   as well as the 'Window Introduction: Command Line' section for more information.

Q2（解答）：
To verify whether the Git system is installed on this computer,
you need to execute a Git command in the 'Command Line' window.

Specifically, using the
'<color=#CF001C>git --version</color>' or '<color=#CF001C>git -v</color>' command
will allow you to confirm that Git has been installed.

Please go to the 'input command field' of the 'Command Line' window
and type
'<color=#CF001C>git --version</color>' or '<color=#CF001C>git -v</color>'.
Note that the letter case must match exactly,
and 'git' must be separated from '--version' by a space.

Once you have finished typing,
press the Enter key or click the arrow icon on the right to execute the command.

   decide which area should be managed by Git,
   and navigate into that folder first.

Q3（解答）：
Navigate to the project path that needs to be managed
to prepare for creating the Git management system.

In this game, you only need to focus on learning Git-related knowledge.
The path in the 'Command Line' will automatically sync with the 'File Manager' window.
(In reality, you would need to use the cd command or other methods to change paths.)

we want to use version control on the project under development.

to sync the 'Command Line' to the target path.

Q4（提示）：
If you are unsure how to create a Git local repository,
here are some hints to help you reach your objective:

1. Make sure you have navigated to the project path you want to manage.
   Creating a repository in the wrong location might track unrelated files.

2. Among Git commands,
   there is a specific command that
   creates a new Git management system at the current command line path.
   Please look it up in the Game Manual for details.

3. Please execute this command in the input command field of the 'Command Line' window.

Q4（解答）：
To create a Git management system in your project to manage it,
you need to execute the '<color=#CF001C>git init</color>' command.

This command will create a new Git management system at the current command line path.
Please make sure the command line path is '<color=#CF001C>Home > Ongoing Project</color>'
so that it won't track unrelated files.

After confirming, go to the input command field of the 'Command Line' window,
type '<color=#CF001C>git init</color>',
and then execute it to create a new Git management system inside 'Ongoing Project'.

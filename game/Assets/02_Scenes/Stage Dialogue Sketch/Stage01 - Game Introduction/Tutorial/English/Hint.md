# Q1 開啟 '檔案管理' 視窗

Q1（Hint）：
Looks like you might be having some trouble opening the 'File Manager' window. Here are some hints to help you reach your objective:

1. Please find the toolbar in the game.
   It is located at the bottom of the screen.

2. Look for the button that opens the 'File Manager' window. You can hover your mouse over the buttons to see what they do.

3. Once you find the button, try clicking it.
   The 'File Manager' window should then open.

Q1（Answer）：
We need to open the 'File Manager' window.
This will allow us to view all the files contained in this simulated computer.
Please find the 'File Manager' button in the toolbar at the bottom of the game.

You can see a button with a folder icon. That is the button to open the 'File Manager' window. Please click it to open the window.

Q2（提示）：
Looks like you are looking for a way to back up your files.
Here are some hints to help you reach your objective:

1. In the 'File Manager' window, find the file you want to back up.
2. Right-click on the file icon to open the context menu.
3. In the context menu, find the 'Copy This File' option and click it.
   This will duplicate the selected file.
4. Please note that the selected file must match the requirements of the current objective.

Q2（解答）：
Backing up your files before making major changes is an excellent habit.
This prevents data loss and makes it easy to restore everything to a previous state.

To successfully back up the 'MidtermReport_v1.ppt' file,we need to find this file in the 'File Manager' window.

Once you find it, please right-click on the file.
This will open a context menu.

There is a 'Copy This File' option in the context menu.Click it to back up the file.

Q3（提示）：
Looks like you need to rename the backed-up file.
Here are some hints to help you reach your objective:

1. Find the backed-up file in the 'File Manager' window.
   Its file name should contain the word <color=#CF001C>(Copy).

2. Click on the copied file to open the 'File Content' window.
3. In this operating system, the renaming feature is located in the 'File Content' window.
   Once you find and click that icon, you should figure out how to rename this file.
4. If you still cannot find the renaming feature, please check the Game Manual.
   The manual introduces the functions of the 'File Content' window in detail.

Q3（解答）：
Renaming backed-up files helps us distinguish between different versions.

To rename the backed-up file,
please find this file in the 'File Manager' window.
Its name is 'MidtermReport_v1(Copy).ppt'.

After clicking it to open the 'File Content' window,
you can rename the file using the pencil icon button in the window's toolbar.

Please click it and select the option for the new name to rename this copied file.

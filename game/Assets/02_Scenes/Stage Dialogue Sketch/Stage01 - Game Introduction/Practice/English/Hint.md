Q4（提示）：
Looks like you are having some trouble adding new content to the file.
Here are some hints to help you reach your objective:

1. Click on the file you want to edit in the 'File Manager' window
   to open the 'File Content' window.

2. There is a plus button at the bottom of the file content.
   Clicking it will add new content.
3. If the current objective is to add content,
   the game will automatically insert the new text based on the story background.

Q4（解答）：

Q7（解答）：
To modify the content in the 'Assignment_Report_v3.docx' file,
please first find and click it in the 'File Manager' window to open the 'File Content' window.

In the 'File Content' window,
the content that needs to be modified
will be marked with <color=#CF001C>(to be modified)</color>.
Therefore, the line we need to modify is:
'<color=#CF001C>Summary: Backing up files is better suited for project management</color>.

There is a pencil icon on the left side of this line.
It is used to modify specific lines of content.
Please click it to modify this line of text we selected.

在上一關中，我們了解了遠端儲存庫的優勢

它可以備份本地儲存庫的開發記錄
同時也能讓您在不同的設備上進行專案開發

此外，它還支援多人共同開發專案
大幅提升了團隊的開發效率

我們在上一關創建了第一個遠端儲存庫
並通過 "git clone 遠端儲存庫網址" 指令
成功地將遠端儲存庫的內容複製到本地電腦

然後，使用 "git remote -v" 指令確認了複製的專案
是否已成功連接到遠端儲存庫

在這個遊戲中
您將會學習到什麼是版本控制
並且在遊戲過程中會使用 Git 指令來進行操作

就算是沒有寫程式的經驗
或者是從未使用過 Git 版本控制工具的人
都可以透過本遊戲來學習這些知識

本遊戲的主要目標是讓您學會：
1. 版本控制的基本概念以及如何有效地應用它
2. 每個 Git 指令的使用方法和用途
3. 透過這些知識來實際管理專案

為了達成上述目標
本遊戲模擬了實際開發專案時
經常會遇到的情境和問題

根據每個情境的難易度和主題的不同
本遊戲設計了多個遊戲關卡

這種方式不僅能幫助您
更有效地學習每個關卡中所要傳達的知識

還能讓您在完成遊戲後
將這些知識實際應用於專案開發中

您使用了 "git add" 指令中的
另一個功能 "<color=#CF001C>git add 資料夾名稱</color>"

透過 "暫存區域" 視窗
您會發現這個指令會將 "資料夾B" 中的所有檔案
全部移動到 "已追蹤" 區域

雖然您也可以使用 "git add 檔案名稱" 
來將資料夾中的檔案一個個地上傳到 "已追蹤" 區域

但是只要掌握了這個指令
您就能夠更加便利地移動檔案

那麼，這次讓我們把 "資料夾B" 中的檔案
全部移出 "已追蹤" 區域吧

這次要透過 "<color=#CF001C>git reset 資料夾名稱</color>"
來進行檔案的移動

和 "git add 資料夾名稱" 的結構相同
"git reset" 也可以在指令前方加入 "資料夾名稱"

讓我再向您介紹幾個在 '命令行' 視窗中非常實用的功能

首先，請注意在輸入欄旁的放大鏡圖標
您可以通過鍵盤上的 Tab 鍵或者點擊圖標
來使用搜尋功能

這對於在還沒有記住 Git 指令
或著是需要快速完成指令時特別有用

另外，在點擊輸入欄後
使用鍵盤上的方向鍵 '上' 和 '下'
來快速填入以前執行過的指令

來創建您的第一個 Git 管理系統

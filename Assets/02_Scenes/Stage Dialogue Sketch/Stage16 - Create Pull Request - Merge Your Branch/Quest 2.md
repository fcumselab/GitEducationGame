在開始創建之前
讓我們先了解一下什麼是 Pull Request

Pull Request 是一個合併請求
它是 Git 服務平台中提供的功能

Pull Request 就和 "git merge" 指令有相同的作用
都是將一個分支合併至另一個分支
但是 Pull Request 主要用於合併遠端的分支

通常，在合併分支的時候
我們只需要根據快速合併或自動合併
來執行操作即可

而 Pull Request 合併請求
需要讓團隊成員查看並審核我們的修改

這樣的方式可以讓團隊參與討論
進一步保證專案的品質和安全性

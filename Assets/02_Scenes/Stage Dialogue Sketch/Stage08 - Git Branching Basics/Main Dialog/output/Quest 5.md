在了解了 "git branch" 指令的功能和輸出結果後
您可能會注意到\r\n"git log" 和 "git branch" 這兩個指令\r\n都能夠查看目前分支的狀態
不過，請您記住：\r\n"git log" 指令主要是用於查看 "提交記錄"\r\n而 "git branch" 指令主要用於查看目前 Git 存在的分支
接下來，如果要在 "new-feature" 分支開發新功能\r\n我們需要先切換到分支上才可以
請您使用 "git checkout 分支名稱" 指令\r\n來切換到 "new-feature" 分支上
